# Dungeon Bot Simulator

## Задание

Реализовать C/C++ программу, которая читает из файла описание подземелья и симулирует работу бота.

Подземелье состоит из комнат с номерами от `0` до `N`. Комната `0` — стартовая. В остальных комнатах могут находиться ресурсы: `iron`, `gold`, `gems`, `exp`. У бота есть запас еды и целевой ресурс, ценность которого в этом запуске удваивается.

Бот работает в две фазы:

1. **Исследование** — тратит половину еды, переходит в непосещённые комнаты по заданным правилам и собирает наиболее ценный ресурс в каждой комнате. Ценность ресурса считается по суммарной стоимости: `количество * стоимость единицы`.
2. **Возвращение** — идёт обратно в комнату `0` кратчайшим путём по посещённым комнатам и, если еды хватает, добирает оставшиеся ресурсы на пути.

Программа должна записать в `result.txt` точную последовательность действий бота: переходы, сбор ресурсов, состояния комнат и итоговую стоимость собранных ресурсов. Если входные данные некорректны, нужно вывести первую некорректную строку.

## Сборка

Проект собирается через `Makefile`.

Из корня проекта:

```bash
make
```
После успешной сборки появится исполняемый файл:
```bash
task
```
Запуск программы:
```bash
./task in.txt
```

Программа читает входные данные из файла, переданного первым аргументом командной строки, и записывает результат в файл:
```bash
result.txt
```
Очистить артефакты сборки можно командой:
```bash
make clean
```

# Тесты

## Запуск

Из папки с проектом:

```bash
make clean
make
python3 run_tests.py ./task --tests tests
```

Скрипт для каждого теста копирует `in.txt` в рабочую папку, запускает программу и сравнивает `result.txt` с `expected.txt` побайтово.
